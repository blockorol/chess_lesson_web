## Chess Lesson Web

Base frontend scaffold for a chess learning website built with Next.js, TypeScript, Tailwind CSS, `chess.js`, `react-chessboard`, and shadcn/ui-friendly setup.

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` - start the local dev server
- `npm run build` - build the production bundle
- `npm run start` - run the production server
- `npm run lint` - run ESLint
